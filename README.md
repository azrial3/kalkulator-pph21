# Kalkulator Pajak PPh 21 (Pasal 17)

Aplikasi berbasis web sederhana untuk menghitung simulasi Pajak Penghasilan (PPh) Pasal 21 berdasarkan tarif terbaru UU HPP (Harmonisasi Peraturan Perpajakan). Pengguna cukup memasukkan nominal PKP (Penghasilan Kena Pajak) dan sistem akan menghitung pajak terutang secara progresif.

## 🚀 Fitur
- **Perhitungan Real-time**: Hasil langsung muncul saat angka diketik.
- **Perhitungan Progresif**: Membagi PKP ke dalam 5 lapisan tarif (5%, 15%, 25%, 30%, dan 35%).
- **Tampilan Responsif**: Nyaman dibuka di perangkat apa pun.
- **Referensi Tarif**: Tabel bantuan untuk melihat ambang batas rentang penghasilan.

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Struktur konten web.
- **CSS3**: Desain antarmuka (Layouting & Styling).
- **JavaScript (Vanilla)**: Logika perhitungan pajak dan manipulasi DOM.
